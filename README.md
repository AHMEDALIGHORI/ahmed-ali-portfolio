## Overview

Ahmed Ali Portfolio is a single-page dark portfolio landing page built with React, Vite, Tailwind CSS, TypeScript, GSAP, Framer Motion, and hls.js.

The page includes a cinematic loading screen, HLS video hero, floating navigation, selected work bento grid, journal pills, scroll-driven parallax gallery, stats, and a video-backed contact footer.

## Features

- Animated loading screen with rotating words and progress counter
- HLS video background powered by `hls.js`
- GSAP entrance animation and ScrollTrigger parallax gallery
- Framer Motion in-view section reveals
- Tailwind CSS theme using custom HSL design tokens
- Responsive layout for mobile, tablet, and desktop
- Animated gradient borders, marquee footer, and smooth-scroll navigation
